# Day 7 : CI + Shell Automation

## Project Overview :
This project demonstrates CI setup using GitHub Actions and Bash Scripting.

## Features :
- CI pipeline that runs on every push
- Displays build time
- Executes system info logging script

## Scripts :
Location: /scrits/system_info.sh

Logs :
- Date
- User
- Uptime
- Disk Usage

## How to Run Locally :
chmod +x scripts/system_info.sh
./scripts/system_info.sh
